Project 1

A/B Testing - Does the new feature of the Cookie Cats game succeed in increasing users' retention?

Code Link

Project Overview

Cookie Cats is a hugely popular mobile puzzle game developed by Tactile Entertainment. It’s a classic “connect three” style puzzle game where the player must connect tiles of the same color in order to clear the board and win the level. It also features singing cats. Not kidding!

As players progress through the game they will encounter gates that force them to wait some time before they can progress or make an in-app purchase, where the first gate in Cookie Cats was moved from level 30 (control group) to level 40 (treatment group). In particular, I analyzed the impact on player retention and game rounds. In this project, since the data I had was not normally distributed, I analyzed the result of an A/B test, which was conducted using Mann Whitney U Test.
