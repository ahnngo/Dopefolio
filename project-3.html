Project 3

Deep Learning - Predicting Student Dropout and Academic Success: an early prediction to help increase students' academic success

Code Link

Project Overview

The graduation rate of my college, Berea College, is only 62.7%, regardless of the fact that all attending students receive full scholarship, which means they have free education, accomodation, food, etc. It triggered my curiosity to create a tool to estimate the risk of dropout and failure based on academic path, demographics, and social-economic factors, helping to provide more accurate help to students by deploying multiple machine learning classification methods and deep learning.

I was successful to increase the accuracy from 0.78 (Random Forest) to 0.92 by implementing efficient feature engineer (eliminating outliers and deleting repeating features) and deploying Deep Learning. To make sure my model was not overfitting the data, I have implemented Dropout and EarlyStopping. There are still works to be done with this project, such as implementing feature selections and testing the model on a dataset collected from my own college, which is already planned to be conducted.
